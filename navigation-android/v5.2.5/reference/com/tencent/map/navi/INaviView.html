Known Indirect Subclasses
CarNaviInfoPanel, CarNaviView
CarNaviInfoPanel 导航默认 UI 组件。

包括：光主图、当前车速、限速和当前路名、退出按钮、设置按钮、 路况、播报、缩放条等。

用户可以通过 CarNaviVIew 控制是否使用默认 UI 组件. 
CarNaviView 驾车导航view。 用于显示默认导航界面，包括地图和导航面板。
