SugOptions setLocation(SearchLatLng location)
设置位置，即定位坐标。

传入后，若用户搜索关键词为类别词（如酒店、餐馆时）， 与此坐标距离近的地点将靠前显示。

当policy=1时，此参数必填。

  • policy=1：出行场景（网约车） – 起点查询。
  • policy=2：出行场景（网约车） – 终点查询。
